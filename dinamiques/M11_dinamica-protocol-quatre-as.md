## Propòsit

Discutir un text a partir de quatre paraules que comencen per la mateixa lletra —en la versió clàssica, Assumptions, Agree, Argue, Aspirations— per estructurar una conversa rica sobre qualsevol tipus de text.

## Objectius

- Estructurar una discussió rica i provocadora sobre un text
- Explicitar supòsits, acords, desacords i aspiracions
- Donar veu ordenada a cada participant abans del debat
- Connectar el text amb les pròpies aspiracions professionals

## Desenvolupament

| Temps | Pas |
|---|---|
| Abans | Presentació del text: lectura centrada en les quatre preguntes «A» (supòsits, acords, desacords, aspiracions) |
| 10' | Supòsits (Assumptions): cadascú n'identifica un (sense discussió) |
| 10' | Acords (Agree): cadascú identifica una part amb què està d'acord |
| 10' | Confrontacions (Argue): cadascú identifica una part amb què discreparia |
| 10' | Aspiracions (Aspirations): cadascú n'identifica una relacionada amb el text |
| 15' | Discussió oberta sobre les troballes, passant d'una «A» a una altra |
| 5' | Anàlisi del procés |

## Materials

- El text a treballar
- Les quatre preguntes «A» visibles
- Full de notes (amb números de pàgina si escau)

## Versió simplificada

Es pot obrir la discussió després de cada «A» en comptes de reservar-la al final, si el grup ho prefereix.

## Indicadors d'observació

- Cada participant aporta a les quatre dimensions
- Les quatre paraules generen una discussió provocadora
- Es respecta el torn sense discutir fins al pas de debat

## Connexió amb el marc

Aquest protocol és una de les estructures de conversa amb què una CPA condueix discussions efectives a partir de textos i idees. La fonamentació (CPA, cicle Abans–Durant–Després, nivells de Guskey) viu al marc [Desenvolupament professional docent](../M11_desenvolupament-professional-docent.md). És una tècnica universal: una CPA d'inspiració ignasiana l'acull en una actitud d'escolta i discerniment compartit, sense que el protocol esdevingui ignasià pel fet d'usar-lo (vegeu [CPA d'inspiració ignasiana](../M11_cpa-inspiracio-ignasiana.md)).
